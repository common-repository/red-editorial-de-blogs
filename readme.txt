Este es un plugin que añade un widget a wordpress que permite mostrar los últimos posts de la Red Editorial de Blogs de manera aleatoria.

== Description ==

La *Red Editorial de Blogs* es un grupo de Blogs que proponen un tema a desarrollar cada semana y cada blog publica su opinión sobre
el tema.

El plugin facilita la tarea de los participantes para añadir vinculos a los demás blogs que participan.

El plugin básicamente añade un widget a la lista de wordpress. El plugin obtiene los últimos posts de la *Red Editorial de Blogs* y
los muestra de manera aleatoria. Por cuestiones de velocidad, el listado no es en tiempo real, sino que se guarda un archivo fijo
que se actualiza cada 10 minutos. Los feeds que deben ser feeds específicos que contengan las entradas de la Red Editorial de Blogs
esto puede hacerse mediante una categoría. Si quieres agregar tu blog al plugin, no dudes en contactarme.




== Installation ==

1. Sube el archivo `red_editorial_blogs.php` a la carpeta `/wp-content/plugins/`
2. Activa el plugin a través del menú 'Plugins' en WordPress
3. Ingresa al menú de 'Widgets' en el que podrás activar el Widget en tu sidebar.
